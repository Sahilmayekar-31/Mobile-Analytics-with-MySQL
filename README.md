# Smartphone Market Analysis with MySQL

This project explores a comprehensive dataset of smartphones using MySQL to uncover pricing trends, feature distributions, and brand performance. It demonstrates SQL proficiency across basic, intermediate, and advanced levels—ideal for business analytics and portfolio presentation.

##  Dataset Overview

The dataset contains detailed specifications for over 500 smartphone models, including:

- Brand, model, price, rating
- Hardware specs: processor, RAM, internal memory, battery
- Features: 5G, NFC, IR blaster, fast charging
- Display: screen size, refresh rate, resolution
- Camera: rear and front megapixels
- OS and extended memory support

##  SQL Techniques Used

-  Basic queries: filtering, sorting, counting
-  Aggregations: `GROUP BY`, `AVG`, `COUNT`
-  Analytical functions: `RANK()`, `SUM() OVER`
-  CTEs for comparative analysis
-  Business logic: price sensitivity, feature combinations, brand performance

##  Key Business Insights

- Top 5 highest-rated smartphones across all brands
- Models under ₹20,000 with premium features (5G, NFC, high battery)
- Most common screen sizes and resolutions
- Brands with the most models and average pricing
- Phones with above-average battery and below-average price
- Top-rated phone per brand
